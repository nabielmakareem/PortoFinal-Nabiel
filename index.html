Tentang Saya

Halo! Saya lulusan baru program studi Teknologi Informasi dari Universitas Muhammadiyah Yogyakarta dengan minat besar pada dunia data. Selama kuliah saya aktif mengerjakan proyek analisis data, mulai dari membersihkan dataset, membuat query SQL, hingga menyajikan hasil dalam bentuk dashboard yang mudah dipahami oleh non-teknis.

Saya percaya data yang baik hanya berguna kalau ceritanya tersampaikan dengan jelas — karena itu saya senang menggabungkan kemampuan analisis statistik dengan storytelling visual. Saat ini saya sedang mencari kesempatan pertama sebagai Data Analyst untuk terus belajar dan memberi dampak nyata lewat data.
